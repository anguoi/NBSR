## schema.json
### 实现功能
- 字段悬浮提示
- 验证字段格式
- 强制要求填写必填项

### 使用方法
> 推荐使用 `Visual Studio Code`

在 json 最前方加入以下内容即可。
```json
"$schema": "https://raw.githubusercontent.com/vhxubo/NBSR/master/schema.json",
```
